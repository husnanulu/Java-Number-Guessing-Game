# Java-Number-Guessing-Game
A logic-based Java application featuring robust exception handling and interactive user feedback.

## Technical Highlights
- **Exception Handling:** Robust management of invalid user inputs (strings, out-of-bounds numbers).
- **Game Logic:** Efficient use of loops and conditional statements for real-time feedback.
- **Randomization:** Utilizing Java's `Random` class for dynamic gameplay.

## How to Run
1. Clone this repository.
2. Open in any Java IDE or compile via terminal.
3. Run the `Main.java` file and start guessing!
